Hola bienvenidos a mi repo de mi aplicacion web de una calculadora solo con puro FRONT-END.

Tecnologias utilizadas:

- JavaScript: Para interaccion dinamica con el usuario, implementando el uso del DOM sin recargar la página.
- CSS: Para el estilo y el aspecto visual de la página web.
- HTML: Para la estructura de la página.
